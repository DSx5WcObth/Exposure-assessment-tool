How to use the program:
1.Unzip all separate zip files (Exposure_assessment_tool_v12.zip.001-005) at once.
2.Open reconstructed EXCEL file by double-clicking.
3.Insert desired data (Processing factor, Residual pesticides name, Concentration of residual pesticides detected in food item [in ppm]) and select Food item no. in STFCJ and National food intake survey for calculation. 
4.Start calculating by clicking "Start calculation" button.
A summary of the calculated results in EXCEL file will automatically transfer to the "Results" folder in the same directly.
